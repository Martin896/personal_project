## Getting Started

### Dependencies 
GCC compiler

#### Installing
```
git clone https://github.com/Martin896/personal_project.git
# clone repository 
```
### Executing program

```bash 
# compile program 
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 file_name

# RESULT 'a.out' file (executable)
``` 
#### Running the program 
```
./a.out

```
